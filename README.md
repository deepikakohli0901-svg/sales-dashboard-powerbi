# 📊 Sales Analytics Dashboard

## 📌 Overview
This project is an interactive Sales Analytics Dashboard built using Power BI. It helps analyze sales performance, profit, revenue, and category-wise trends to support better business decisions.

---

## 🎯 Objective
To analyze sales data and generate meaningful insights like revenue growth, profit trends, and category performance to improve business strategy.

---

## 🛠️ Tools Used
- Power BI Desktop
- Microsoft Excel (Dataset)
- DAX (Data Analysis Expressions)

---

## 📊 Key Insights
- Total Sales and Revenue
- Total Profit and Profit Margin
- Category-wise Sales Analysis
- Region/State-wise Performance
- Monthly/Yearly Sales Trends
- Top Selling Products
